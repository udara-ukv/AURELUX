## üìû Support & Help

### If Something Doesn't Work:

1. **Check Console:** Press F12, check for errors
2. **Clear Cache:** Ctrl+Shift+Delete, clear browser cache
3. **Restart Server:** Stop and restart the HTTP server
4. **Port In Use:** Try different port: `python3 -m http.server 9000`

### Common Issues:

**Issue:** Pages not loading
- **Solution:** Ensure you're in `/home/udara/Documents/my-webapp` directory

**Issue:** Styles not showing
- **Solution:** Clear browser cache (Ctrl+F5)

**Issue:** Cart not persisting
- **Solution:** Check if localStorage is enabled
